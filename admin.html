Cloudinary Setup Guide

To enable File Uploads, you need an unsigned upload preset:

  1. Log in to Cloudinary Console.
  2. Go to Settings (gear icon) > Upload tab.
  3. Scroll to Upload presets and click Add upload preset.
  4. Change "Signing Mode" to Unsigned.
  5. Copy the generated Preset Name (e.g., ml_default).
  6. Paste your Cloud Name and Preset Name in the builder.

Unsigned presets allow direct uploads from users without exposing your API credentials.
